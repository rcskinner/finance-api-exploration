# Finance API Exploration 

Small repository exploring different finance API options available within python, evaluating the use-cases and pros/cons of each. 

Evaluating options shown in this [article](https://towardsdatascience.com/best-5-free-stock-market-apis-in-2019-ad91dddec984)

1. Yahoo Finance (Recently deprecated & revived)
2. Google Finance (via Google Sheets)
3. IEX Cloud 
4. Alphavantage 
6. World Trading Data
